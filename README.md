# 项目简介

用来生成 Surge 配置文件的 `python` 脚本。

# 实现功能

- [x] 支持不同类型的节点订阅链接（Surge、SS）
- [x] 支持同时输入多个节点订阅链接（相同类型或者不同类型）
- [x] 支持自定义规则链接
- [x] 支持为策略组添加节点
- [x] 支持修改策略组名字（有限）

# 使用方法

- 下载或者克隆项目后，进入项目文件夹：

  ```bash
  # 例如
  cd ConfigSplicing
  ```

- 输入命令：

  ```bash
  python main.py
  ```

- 然后根据提示做完就可以了，能不能成功就随缘吧。:relieved:
